# 클라이언트 관점 앱 평가 (2026-05-12)

## 이 앱은 무엇인가

나라장터 공공입찰 공고를 자동 수집하고, 낙찰 이력과 AI 입찰가 추천을 제공하는 SaaS 플랫폼.  
주 타깃: 공공 조달 입찰에 참여하는 중소 기업의 입찰 담당자.

---

## 실제로 작동하는 것 (강점)

| 기능 | 평가 |
|------|------|
| 키워드·카테고리·마감일 필터 검색 | ✅ 실용적. 공고 수천 건 중 필요한 것만 빠르게 좁힘 |
| 낙찰 결과 + 참여사 이력 | ✅ "이 공고에 누가 얼마에 입찰했나" 즉시 확인 가능 |
| AI 입찰가 추천 (보수/표준/공격적) | ✅ 직접 계산하던 작업을 자동화. 핵심 차별점 |
| 마감 임박 알림 (이메일) | ✅ 놓치기 쉬운 공고 추적 가능 |
| 데이터 최신성 (매일 09:05 자동 수집) | ✅ GitHub Actions로 안정화 완료 |
| 분석·보고서 (PDF/CSV 내보내기) | ✅ 기관별·산업별 낙찰 트렌드 시각화 |

**경쟁 우위:** 낙찰 이력 + AI 가격 추천 조합은 입찰 담당자가 직접 구성하기 어렵다.

---

## 개선이 필요한 것 (우선순위 순)

### 🔴 즉시 수익·이탈에 영향

**1. AI 추천을 체험하기 전에 결제를 요구**  
- 가장 강력한 기능이 Pro 장벽 뒤에 숨겨져 있음  
- 해결: 월 3회 무료 체험 제공 → 가치를 느낀 후 업그레이드 유도

**2. Free 제한이 너무 빠르게 막힘**  
- 즐겨찾기 50개, 알림 3개 → 실제 입찰 담당자는 10~20개 공고 동시 추적이 일반적  
- 제한에 걸리면 "쓸 수 없는 앱"으로 인식하고 이탈  
- 해결: 즐겨찾기 100개, 알림 5개로 완화

**3. 카카오 알림 미구현**  
- 한국 B2B 환경에서 이메일보다 카카오톡 알림이 실용적  
- 코드 구조는 있지만 실제 미동작 (빈 껍데기)

### 🟠 유료 전환율에 영향

**4. 팀 기능이 Enterprise(₩199k)에만 포함**  
- 2~3인 소규모 팀은 Pro(₩49k)면 충분한데 ₩199k 강제  
- 해결: 팀원 2명까지는 Pro에 포함

**5. OAuth 로그인 없음**  
- 이메일+비밀번호만 지원 → 구글/카카오 간편 로그인 부재로 가입 마찰 증가

### 🟡 장기적으로

**6. 검색 Boolean 연산자 없음**  
- `"SW개발" AND NOT "유지보수"` 같은 조건 불가. 고급 사용자 요구 충족 어려움

**7. 보고서 자동 발송 없음**  
- PDF 내보내기는 있지만 주간 요약을 이메일로 자동 수신하는 기능 없음
