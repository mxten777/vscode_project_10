# Archive

현재 의사결정에 바로 필요하지 않은 문서를 모아 둔 보관 영역입니다.

## 언제 보나

- 특정 날짜의 작업 내역이 필요할 때
- 과거 상태 보고서나 테스트 스냅샷을 비교할 때
- 현재 문서가 왜 이렇게 정리됐는지 배경이 필요할 때

## 폴더 설명

- `history/`: 세션 로그, 작업 일지, 연속 작업 기록
- `reports/`: 당시 시점 기준 상태 보고서, 테스트 리포트, 스냅샷 문서

## 원칙

- 현재 판단 기준은 여기 문서보다 `docs/START_HERE.md` 와 `docs/READINESS_REPORT.md` 가 우선입니다.
- 배포/DB 기준은 `supabase/migrations/*` 와 최신 운영 문서를 우선합니다.
- 아카이브 문서는 삭제 대신 참조용으로만 유지합니다.
