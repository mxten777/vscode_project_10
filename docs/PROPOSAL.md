## 제안 요약

귀사의 공공입찰 업무 효율을 높이고, 중요한 공고를 놓치지 않도록  
**AI 기반 공공입찰 지원 서비스 BidSight** 도입을 제안드립니다.

---

## 1. 현황 진단 — 귀사가 겪고 있는 문제

공공입찰에 참여하는 대부분의 기업은 아래 문제를 공통으로 경험합니다.

### 문제 1. 매일 반복되는 수작업 모니터링

나라장터(G2B)에는 하루 평균 3,000건 이상의 공고가 게시됩니다.  
담당자가 매일 직접 접속·검색·확인하는 데 **1~3시간**이 소요됩니다.

> "담당자가 매일 오전 2시간은 나라장터 뒤지는 데 써요."  
> — IT 솔루션 기업 입찰 담당자

### 문제 2. 중요한 공고 누락

수작업 모니터링은 완전하지 않습니다.  
마감 임박 공고를 뒤늦게 발견하거나, 적합한 공고를 아예 놓치는 일이 반복됩니다.

> "마감 하루 전에 공고를 발견해서 놓친 적이 한두 번이 아닙니다."  
> — 동일 기업 담당자

### 문제 3. 시장 인텔리전스 부재

- 어느 기관이 우리 업종을 자주 발주하는지 파악하기 어렵습니다.
- 낙찰 이력과 경쟁 현황을 분석할 도구가 없습니다.
- 입찰 전략을 데이터가 아닌 경험과 감으로 수립합니다.

## 3. 주요 기능 상세

### 3.1 공고 자동 수집 및 검색

- 나라장터 공공데이터포털 공식 API로 매일 자동 수집
- 키워드 / 기관 / 업종 / 공고 상태 / 예산 범위로 복합 필터링
- 마감임박순·예산규모순 정렬

### 3.2 복합 조건 알림

- 키워드 + 기관 + 업종 + 예산 조건을 조합하여 알림 규칙 등록
- 조건 매칭 공고 발생 시 즉시 이메일 발송
- 규칙별 발송 이력 조회

### 3.3 즐겨찾기

- 관심 공고 즐겨찾기 등록·해제·일괄 관리

### 3.4 분석 대시보드

- 기간별 공고 건수·예산 추이
- 발주 기관 TOP 10 / 업종 TOP 10
- 상태별 분포 (진행중 / 마감 / 결과발표)

### 3.5 AI 낙찰률 예측 (2026년 하반기 오픈 예정)

- 과거 낙찰 데이터 학습 기반 예측 모델
- 기관·업종·예산 조건별 낙찰 가능성 수치 제공
