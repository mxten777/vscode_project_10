## 1. 왜 BidSight인가

### 입찰 실무의 반복 문제

공공입찰을 담당하는 실무자는 매일 같은 작업을 반복합니다.

- 나라장터에서 관련 공고를 직접 검색하고 훑는 시간 (1~3시간/일)
- 마감 임박 공고를 뒤늦게 발견하거나 놓치는 경험
- 어떤 공고부터 검토해야 할지 기준이 없어 시간을 허비하는 상황
- 기관별·업종별 시장 흐름을 파악할 도구가 없어 감으로만 판단하는 상황

BidSight는 이 반복 작업과 판단 부담을 줄이는 데 집중합니다.

### 기존 서비스와 무엇이 다른가

| 구분 | 나라장터 포털 | 기존 알림 서비스 | BidSight |
|------|------------|----------------|---------|
| 공고 수집 | 직접 검색 | 이메일 알림만 | 자동 수집 + 맞춤 검색 |
| 알림 조건 | 없음 | 키워드 1개 | 키워드+기관+업종+예산 복합 조건 |
| 분석 리포트 | 없음 | 없음 | 기관별·업종별 통계 대시보드 |
| UI/UX | 구형 포털 | 이메일 텍스트 | 모던 SaaS (웹·모바일) |
| 월 비용 | 무료 | 1~15만 원 | 무료~49,000원 |

## 2. 서비스 구성

### 2.1 공고 자동 수집 및 검색

나라장터(G2B) 공고를 매일 자동으로 수집하여 검색 가능한 형태로 정리합니다.

- **키워드 검색**: 공고명, 기관명, 업종 키워드로 즉시 검색
- **복합 필터**: 발주기관 / 업종·공종 / 공고 상태 / 예산 범위 / 마감일
- **정렬**: 최신순 / 마감임박순 / 예산 규모순
- **페이지네이션**: 20건/페이지, 빠른 목록 탐색

### 2.2 맞춤 알림

중요한 공고를 놓치지 않도록 조건 기반 알림을 제공합니다.

- **알림 규칙 등록**: 키워드 또는 복합 조건(기관+업종+예산)으로 규칙 설정
- **이메일 알림**: 조건 매칭 공고 발생 시 즉시 이메일 발송
- **알림 이력**: 발송된 알림 내역 조회
- **카카오 알림톡**: 2026년 하반기 예정

### 2.3 즐겨찾기 관리

관심 공고를 체계적으로 관리합니다.

- 공고 즐겨찾기 등록 / 해제
- 즐겨찾기 목록 일괄 조회 및 관리
- Pro 플랜 기준 사실상 무제한 등록

### 2.4 분석 대시보드 및 리포트

데이터 기반 입찰 전략 수립을 지원합니다.

- **기간별 통계**: 공고 건수, 총 예산 규모, 참여 기관 수
- **상태별 분포**: 진행중 / 마감 / 결과발표 비율
- **발주 기관 TOP 10**: 자주 발주하는 기관 순위
- **업종별 TOP 10**: 발주 규모 큰 업종 순위

### 2.5 AI 낙찰률 예측 (준비 중)

과거 낙찰 데이터를 학습한 AI 모델이 참여 전 낙찰 가능성을 예측합니다.

- 기관·업종·예산 조건 기반 낙찰률 추정
- 유사 공고 낙찰 이력 비교
- 현재 AI 서비스(bid-ai-service) 구현 완료, 운영 연결 준비 중

## 3. 사용 흐름

```
[1단계] 시스템이 공고를 자동 수집
         ↓
[2단계] 키워드·기관·예산으로 필요한 공고를 빠르게 좁힘
         ↓
[3단계] 중요 공고는 즐겨찾기 또는 알림 규칙으로 등록
         ↓
[4단계] 낙찰 데이터·분석 화면으로 참여 여부 판단
         ↓
[5단계] 팀이 같은 화면·기준으로 검토 결과 공유
```

## 6. 기술 기반

BidSight는 안정성과 보안을 중심으로 설계된 클라우드 네이티브 서비스입니다.

- **데이터 소스**: 나라장터(G2B) 공공데이터포털 공식 API 연동
- **인프라**: Vercel(배포) + Supabase Seoul 리전(데이터베이스)
- **보안**: Row Level Security(RLS)로 조직별 데이터 완전 격리
- **가용성**: 99.9% SLA 목표 (Enterprise 플랜)
- **AI 서비스**: FastAPI 기반 낙찰률 예측 서비스 (Railway 운영)

## 8. 자주 묻는 질문

**Q. 나라장터와 공식 연동인가요?**  
A. 공공데이터포털 공식 API를 통해 연동합니다. 스크래핑이 아닌 공식 데이터 수신 방식입니다.

**Q. 데이터는 얼마나 자주 갱신되나요?**  
A. 현재 1일 1회 수집하며, 향후 10분 단위 수집으로 고도화할 예정입니다.

**Q. 모바일에서도 사용할 수 있나요?**  
A. 웹 기반 서비스로, 모바일 브라우저에서 접속 가능합니다.

**Q. Free 플랜에서 Pro로 전환 시 기존 데이터가 유지되나요?**  
A. 즐겨찾기, 알림 규칙, 히스토리 모두 그대로 유지됩니다.

**Q. 보안은 어떻게 관리되나요?**  
A. 조직별 데이터 격리(RLS), 전송 암호화(TLS), 저장 암호화가 적용됩니다.
