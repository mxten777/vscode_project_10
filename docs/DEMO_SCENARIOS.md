## 2. 추천 시연 순서

1. 랜딩에서 서비스 가치 설명
2. 메인 검색창에 `플랫폼` 입력
3. OPEN 공고 여러 건을 비교하며 선별 흐름 설명
4. 결과 공고 `경기도 공공빅데이터 플랫폼 구축 1단계` 상세 진입
5. 즐겨찾기 또는 알림 시나리오로 `AI` 키워드 관심군 설명
6. 분석 대시보드/리포트로 마무리

## 3. 말할 때 주의할 점

- 검색 결과 수 자체보다 선별 속도를 강조합니다.
- 상세 화면에서는 "판단 근거" 를 강조하고, AI 자동 추천처럼 과장하지 않습니다.
- 알림 설명에서는 "좋은 공고를 찾는 것" 보다 "놓치지 않는 구조" 를 강조합니다.
