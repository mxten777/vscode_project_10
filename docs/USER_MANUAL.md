## 1. 시작하기

### 1.1 플랫폼 소개

BidSight는 **대한민국 나라장터(조달청)** 의 공공 입찰 공고를 실시간으로 수집·분석하여, 기업이 원하는 공고를 빠르게 찾고 맞춤 알림을 받을 수 있도록 도와주는 AI 기반 SaaS 플랫폼입니다.

| 항목 | 내용 |
|------|------|
| 서비스 URL | https://bid-platform.vercel.app |
| 지원 브라우저 | Chrome 115+, Edge 115+, Safari 16+, Firefox 115+ |
| 권장 해상도 | 1280 × 800 이상 |
| 모바일 지원 | 반응형 웹 (iOS Safari, Android Chrome) |

### 1.2 주요 기능 요약

| 기능 | 설명 |
|------|------|
| 공고 검색 | 키워드·기관·업종·예산·기간 복합 필터 검색 |
| D-day 표시 | 마감 임박 공고 자동 강조 표시 |
| 즐겨찾기 | 관심 공고를 북마크하여 빠르게 재접근 |
| 알림 규칙 | 조건 기반 자동 알림 설정 (이메일·카카오) |
| 리포트 | 업종별·기관별 낙찰 통계 분석 |

## 2. 회원가입 및 로그인

### 2.1 회원가입

1. 브라우저에서 **https://bid-platform.vercel.app** 에 접속합니다.
2. 화면 우상단 **[로그인]** 버튼을 클릭합니다.
3. 로그인 페이지 하단 **"계정이 없으신가요? 회원가입"** 링크를 클릭합니다.
4. 이메일 주소와 비밀번호(6자 이상)를 입력한 후 **[가입하기]** 버튼을 클릭합니다.
5. 입력한 이메일로 인증 메일이 발송됩니다. 메일함에서 인증 링크를 클릭하면 가입이 완료됩니다.

> ⚠️ **주의**: 회원가입 후 이메일 인증을 완료해야 로그인이 가능합니다. 인증 메일이 오지 않으면 스팸 폴더를 확인해 주세요.

### 2.2 로그인

1. 로그인 페이지에서 **이메일**과 **비밀번호**를 입력합니다.
2. **[로그인]** 버튼을 클릭합니다.
3. 로그인 성공 시 메인 대시보드로 자동 이동합니다.

### 2.3 로그아웃

- 헤더 우상단 **프로필 아이콘** → **[로그아웃]** 클릭

### 2.4 비밀번호 변경

1. 헤더 우상단 **프로필 아이콘** → **[비밀번호 변경]** 클릭
2. 새 비밀번호를 두 번 입력합니다.
3. **[변경]** 버튼 클릭 시 즉시 반영됩니다.

## 3. 메인 대시보드 (공고 검색)

### 3.1 화면 구성

```
┌─────────────────────────────────────────────────────────┐
│  헤더 (로고 / 네비게이션 / 테마전환 / 프로필)              │
├─────────────────────────────────────────────────────────┤
│  히어로 배너 (통계 요약: 전체공고 / 마감임박 / 집계)        │
├─────────────────────────────────────────────────────────┤
│  카테고리 칩 (전체 / 소프트웨어 / 건설 / 용역 / 물품 / 시설) │
├─────────────────────────────────────────────────────────┤
│  트렌딩 키워드 (AI 플랫폼 / 정보화시스템 / 클라우드 등)     │
├─────────────────────────────────────────────────────────┤
│  검색·필터 패널                                           │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 키워드 입력 │ 기관 │ 상태 │ 예산min~max │ 기간 │   │
│  └─────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────┤
│  공고 카드 목록 (D-day 뱃지 / 즐겨찾기 버튼)               │
├─────────────────────────────────────────────────────────┤
│  페이지네이션                                             │
└─────────────────────────────────────────────────────────┘
```

### 3.2 검색 방법

#### 키워드 검색
- 검색창에 원하는 키워드(공고명, 기관명 등)를 입력하고 **Enter** 또는 검색 아이콘을 클릭합니다.
- 예: `AI 플랫폼`, `정보화시스템`, `전산 유지보수`

#### 카테고리 필터
- 히어로 배너 하단의 **카테고리 칩**을 클릭하여 업종별로 필터링합니다.

| 칩 | 포함되는 공고 |
|----|--------------|
| 전체 | 모든 공고 |
| 소프트웨어 | SW개발, 정보화시스템, AI 플랫폼 등 |
| 건설·공사 | 건설, 공사, 시공 등 |
| 용역·서비스 | 용역, 컨설팅, 위탁 등 |
| 물품·장비 | 구매, 납품, 장비 등 |
| 시설관리 | 시설, 유지보수, 청소 등 |

#### 고급 필터 (검색 패널)
- **기관명**: 특정 발주 기관만 표시
- **상태**: `진행중 (OPEN)` / `마감 (CLOSED)` / `낙찰완료 (RESULT)`
- **예산 범위**: 최소~최대 금액(만원) 설정
- **공고 기간**: 시작일~종료일 범위 설정

> 💡 **팁**: 여러 필터를 동시에 적용할 수 있습니다. 필터 패널 상단 **[초기화]** 버튼으로 모든 조건을 한 번에 해제합니다.

#### 트렌딩 키워드
- 히어로 배너 내 플로팅 키워드 pill을 클릭하면 해당 키워드로 즉시 검색됩니다.

### 3.3 공고 카드 읽기

각 공고 카드에는 다음 정보가 표시됩니다:

| 항목 | 설명 |
|------|------|
| 공고명 | 입찰 공고 제목 (최대 2줄) |
| 발주기관 | 공고를 게시한 기관명 |
| 예산 | 추정 예산 금액 |
| 마감일 | 입찰 마감 날짜 |
| D-day 뱃지 | 🔴 D-3 이내: 긴급 / 🟠 D-7 이내: 주의 / 없음: 여유 |
| 상태 뱃지 | `진행중` / `마감` / `낙찰완료` |
| ⭐ 즐겨찾기 | 클릭 시 즐겨찾기 추가/제거 |

### 3.4 페이지네이션

- 한 페이지에 **최대 10건**의 공고가 표시됩니다.
- 하단 페이지 번호 또는 **이전/다음** 버튼으로 이동합니다.
- 현재 페이지: 파란색 강조 표시

## 4. 공고 상세 조회

공고 카드 클릭 시 상세 페이지로 이동합니다.

### 4.1 기본 정보

| 항목 | 설명 |
|------|------|
| 공고명 | 전체 입찰 공고 제목 |
| 발주기관 | 기관명 및 담당 부서 |
| 공고번호 | 나라장터 고유 번호 |
| 추정 예산 | 금액 (원 단위 표시) |
| 마감일시 | 날짜 및 시간 |
| 공고 분류 | 업종 카테고리 |

### 4.2 D-day 카운트다운

- 마감일이 **3일 이내**: 🔴 빨간색 긴급 카드 표시
- 마감일이 **7일 이내**: 🟠 주황색 카드 표시
- 마감일이 지난 경우: 마감 처리 표시

### 4.3 진행 타임라인

공고의 진행 단계를 4단계 타임라인으로 표시합니다:

```
① 공고 등록  →  ② 입찰 진행  →  ③ 마감  →  ④ 결과 발표
  (등록일)       (OPEN 상태)     (마감일)    (낙찰결과 있을 때)
```

- ✅ **완료**: 이미 지난 단계
- 🔵 **현재**: 현재 진행 중인 단계
- ⭕ **대기**: 아직 진행되지 않은 단계

### 4.4 낙찰 정보

공고 상태가 `낙찰완료(RESULT)`인 경우 낙찰 기업명, 낙찰 금액, 낙찰 일자가 표시됩니다.

### 4.5 공고 원문 링크

- **[나라장터에서 보기]** 버튼을 클릭하면 조달청 나라장터 원문 페이지로 이동합니다.

## 5. 즐겨찾기

### 5.1 즐겨찾기 추가

- **공고 카드** 우상단 ⭐ 아이콘 클릭
- 또는 **공고 상세 페이지** 내 [즐겨찾기 추가] 버튼 클릭

> 노란색 별 아이콘 = 즐겨찾기 등록됨

### 5.2 즐겨찾기 목록 조회

- 상단 네비게이션 **[즐겨찾기]** 메뉴 클릭
- 등록된 공고 건수가 헤더에 표시됩니다.

### 5.3 즐겨찾기 제거

- 즐겨찾기 목록에서 해당 공고의 ⭐ 아이콘 재클릭
- 또는 즐겨찾기 페이지 내 [제거] 버튼 클릭

> 즐겨찾기 데이터는 계정에 연결 저장되므로, 다른 기기에서 로그인해도 동일하게 표시됩니다.

## 6. 알림 관리

### 6.1 알림이란?

알림 규칙을 등록하면, 해당 조건에 맞는 새 공고가 올라올 때 자동으로 이메일 또는 카카오 알림을 받을 수 있습니다.

### 6.2 알림 규칙 생성

1. 상단 네비게이션 **[알림 관리]** 클릭
2. 화면 우상단 **[+ 규칙 추가]** 버튼 클릭
3. 아래 항목을 입력합니다:

| 항목 | 설명 | 예시 |
|------|------|------|
| 규칙 이름 | 알림 규칙의 이름 | `AI 플랫폼 공고 알림` |
| 키워드 | 공고명에 포함되어야 할 단어 | `AI`, `클라우드`, `정보화` |
| 기관 | 특정 기관만 알림받을 경우 기관명 | `서울특별시` |
| 최소 예산 | 예산 하한 금액 (만원) | `5000` |
| 최대 예산 | 예산 상한 금액 (만원) | `50000` |
| 알림 채널 | `이메일` / `카카오` / `둘 다` | 이메일 |

4. **[저장]** 버튼 클릭

### 6.3 알림 규칙 관리

- **활성화/비활성화**: 규칙 카드의 토글 스위치로 즉시 전환
- **수정**: 규칙 카드 **[편집]** 버튼
- **삭제**: 규칙 카드 **[삭제]** 버튼

### 6.4 알림 발송 이력

- 알림 관리 페이지 하단에서 최근 알림 발송 이력을 확인할 수 있습니다.
- 발송 일시, 규칙명, 매칭된 공고명이 표시됩니다.

### 6.5 알림 채널 설정

| 채널 | 설명 |
|------|------|
| 이메일 | 가입 시 등록한 이메일로 자동 발송 |
| 카카오 | 카카오 알림톡 (별도 연동 필요, 추후 제공 예정) |

## 7. 리포트

### 7.1 리포트 개요

리포트 페이지에서는 나라장터 공고 데이터를 기반으로 한 **통계 분석**을 제공합니다.

### 7.2 제공 통계

| 섹션 | 내용 |
|------|------|
| 전체 요약 | 총 공고 수, 발주 기관 수, 집계 업종 수 |
| 월별 공고 추이 | 월별 공고 건수 변화 차트 |
| 업종별 분포 | 카테고리별 공고 비율 |
| 기관별 현황 | 발주 기관별 공고 건수 상위 목록 |
| 예산 구간 분포 | 예산 규모별 공고 현황 |
| 낙찰 기업 분석 | 자주 낙찰되는 기업 현황 |

### 7.3 기간 필터

- 리포트 우상단 기간 선택 드롭다운으로 **최근 1개월 / 3개월 / 6개월 / 1년** 단위로 조회할 수 있습니다.

## 9. 자주 묻는 질문 (FAQ)

**Q1. 공고 데이터는 얼마나 자주 업데이트되나요?**  
A. 나라장터 API 연동을 통해 **매시간** 자동 수집됩니다. 실시간 반영이 필요한 경우 브라우저를 새로고침하세요.

**Q2. 알림은 언제 발송되나요?**  
A. 새 공고 수집 후 알림 규칙과 매칭되는 즉시 처리됩니다. 최대 **1시간** 이내 발송됩니다.

**Q3. 즐겨찾기 개수 제한이 있나요?**  
A. 현재 버전에서는 개수 제한이 없습니다.

**Q4. 알림 규칙은 몇 개까지 만들 수 있나요?**  
A. 계정당 최대 **20개**의 알림 규칙을 등록할 수 있습니다.

**Q5. 공고 원문을 보려면 어떻게 하나요?**  
A. 공고 상세 페이지 하단의 **[나라장터에서 보기]** 버튼을 클릭하면 조달청 나라장터 원문 페이지로 이동합니다.

**Q6. 모바일에서도 사용할 수 있나요?**  
A. 반응형 웹으로 설계되어 스마트폰·태블릿에서도 이용 가능합니다. 모바일 앱은 추후 출시 예정입니다.

**Q7. 비밀번호를 잊어버렸어요.**  
A. 로그인 페이지에서 **[비밀번호 찾기]** 링크를 클릭하면 이메일로 재설정 링크가 발송됩니다.

**Q8. 공고가 검색이 안 되어요.**  
A. 키워드가 공고명과 정확히 일치하지 않을 수 있습니다. 짧은 키워드나 다른 유사어로 다시 시도해 보세요.
